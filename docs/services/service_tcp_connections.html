Сервис tcp_connections()

Описание

Служебный трафик в сети BitPie.NET может передавться по стандартному протоколу TCP. Сетевая служба tcp_connections() позволяет включать/отключать использование TCP соединений во всех модулях ПО. В настройках службы пользователь может установить номер TCP порта для приема входящих соединений от других узлов. Утилита miniupnpc используется для настройки UPnP устройств и автоматической настройки port-mapping.

Зависит от

  • network()

Влияет на

  • tcp_transport()
  • identity_propagate()
  • identity_server()

Настройки сервиса

  • services/tcp-connections/enabled - включение/выключение сервиса tcp_connections()
  • services/tcp-connections/tcp-port - установка номера порта для входящих TCP соединений
