Словарь понятий

BitPie.NET

глобальная сеть для распределенного хранения и обмена данными

Программа BitPie.NET

программное обеспечение предоставляемое и распространяемое BitPie.NET Inc. под частной лицензией с условно-открытым исходным кодом - доступным для свободного скачивания и просмотра

Узел

компьютер, подключенный к сети Интернет, на котором установлено ПО BitPie.NET

Сеть BitPie.NET

добровольное объединение людей, использующих свои личные компьютеры как отдельные взаимодействующие друг с другом узлы BitPie.NET

Данные

любая информация, которую можно хранить в виде файла на жестком диске компьютера

Резервная копия

точная копия данных принадлежащих определенному пользователю, созданная в некий момент времени

Резервная копия папки

это резервная копия данных, полученных архивированием всех файлов, хранящихся в данной папке на жестком диске компьютера, включая (по выбору пользователя) и все подпапки

Распределенная копия

это резервная копия, которая хранится на различных узлах в сети BitPie.NET

Блок

файл на жестком диске содержащий лишь часть резервной копии и имеющий порядковый номер, присваиваемый в момент создания блока

Кусок

часть блока, полученная в процессе создания резервной копии и имеющая порядковый номер - позицию в блоке

Размер резервной копии

суммарный размер всех блоков относящихся к указанной резервной копии

Пакет

Служебный пакет

полезная информация, передаваемая от одного узла к другому в сети BitPie.NET и содержащая электронную подпись отправителя

Пакет данных

Порция данных

это пакет, который содержит зашифрованный кусок данных

Хранение данных в сети BitPie.NET

методика обработки информации при которой исходные данные шифруются, архивируются и размещаются на разных узлах, разбитые на блоки и куски

Восстановление

процесс обратного преобразования данных из отдельных кусков и воссоздания исходной информации из существующей резервной копии

Загрузка в сеть BitPie.NET

процесс создания распределенной копии данных, порции которой размещены на хранителях пользователя

Скачивание из сети BitPie.NET

процесс восстановления данных из распределенной копии, которая была предварительно размещена на хранителях пользователя

Хранитель

узел в сети BitPie.NET, чей владелец добровольно согласился хранить данные других пользователей на жестком диске своего персонального компьютера

Клиент

узел в сети BitPie.NET, владелец которого использует пространство на жестком диске других узлов для хранения своих данных

Папка клиентов

папка на жестком диске пользователя, где хранятся данные его клиентов

Отданное пространство хранителя

максимальный размер, который может достигать папка клиентов

Использованное пространство хранителя

текущий размер папки клиентов

Выделенное пространство клиента

значение в байтах, заранее согласованное с пользователем, которое задает максимальный общий объем всех данных, которые клиент может разместить на данном узле

Свободное пространство хранителя

разность между отданным пространством и общим объемом выделенного пространства для всех клиентов пользователя

Использованное пространоство клиента

суммарный размер всех уже созданных распределенных копий данного пользователя

Запрошенное пространство клиента

верхняя граница использованного пространоства клиента для данного пользователя - другими словами максимальный общий объем данных, которые могут быть размещены на хранителях

Свободное пространство клиента

разность между запрошенным и использованным пространоством клиента

Команда

пакет данных содержащий лишь короткую кодовую фразу, сообщение или другую информацию, используемую для взаимодействия узлов в сети

Сообщение

зашифрованное текстовое сообщение, адресованное конкретному пользователю

Шифрование

преобразование данных с целью их защиты

Архивация

преобразование данных, которые могут находится в разных местах, с целью создания единого потока и уменьшения их общего размера

Identity

общедоступный файл содержащий открытый ключ, цифровую подпись, контактные данные и другую информацию принадлежащую определенному пользователю

Identity сервер

сервер принимающий и хранящий файлы Identity

Identity URL

IDURL

Идентификатор пользователя

уникальный адрес файла Identity в интернете

Вход в сеть

процесс создания нового файла Identity и отправка его на несколько Identity серверов и получение уникального IDURL адреса

Восстановление Identity

процесс загрузки Identity с одного из Identity серверов (по известному IDURL) и последующей проверки соответствия содержимого файла и закрытого ключа пользователя
