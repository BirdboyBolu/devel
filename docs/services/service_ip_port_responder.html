Сервис ip_port_responder()

Описание

Сетевая служба ip_port_responder() позволяет другим участникам сети производить процедуру STUN, для определения собственного IP адреса и номра открытого UDP порта.

Она работает как простейший STUN сервер и отвечает на запрросы других узлов, у которых активна служба my_ip_port(). Обе эти службы работают в паре, но на разных машинах в сети.

Для поддержки других пользователей сети BitPie.NET настоятельно рекомендуется держать службу ip_port_responder() активной, она включена по умолчанию.

Зависит от

  • udp_datagrams()
  • entangled_dht()

Настройки сервиса
